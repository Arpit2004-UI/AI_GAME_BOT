# 🎮 AI Game Bot – Tic Tac Toe

An advanced Tic Tac Toe game powered by Artificial Intelligence using **Minimax Algorithm** and **Alpha-Beta Pruning**, built with Python and Pygame.

---

## 🚀 Features

- 🤖 Play against AI (X or O)
- 🎯 3 Difficulty Levels:
  - Easy (Random + shallow search)
  - Medium (Limited depth)
  - Hard (Optimal AI)
- 🧠 Algorithms:
  - Minimax
  - Alpha-Beta Pruning
- 📊 Live Decision Tree Visualization
- 📈 Performance Metrics:
  - Nodes Evaluated
  - Branches Pruned
  - Search Depth
  - Think Time
  - AI Confidence Score
- 💡 Hint System (suggest best move)
- 🏆 Score Tracking (Win / Loss / Draw)

---

## 🧠 Algorithms Used

### 1. Minimax
- Explores all possible game states
- Guarantees optimal move
- Time complexity: O(b^d)

### 2. Alpha-Beta Pruning
- Optimized Minimax
- Cuts unnecessary branches
- Improves performance significantly
